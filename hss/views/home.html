Value Glossary

ℕ
natural numbers (e.g. 0, 1, 2, 3, etc.)
ℤ
integers (equivalent to ℕ and -ℕ)
<numbers>
represents rational numbers (e.g. 1, 0.1, -1)
<object>
represents the object type(s) accepted in the current context and the object names of this/these type(s).

Reminder

All properties accept the special keywords (default, inherit* and current).

* except properties set on the root element.
