    • AXR stands for Arbitrary XML Rendering. It's aim is to provide a better alternative to HTML+CSS It uses XML for the content and HSS for the design and simple behavior of the interface. HSS is a language based on CSS, but offers many more advanced features, such as object orientation, rule nesting, expressions, references to other objects, modularization (code reuse), etc. JavaScript will be used as well for advanced behavior.
